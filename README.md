# Riddhi Shobhavat Portfolio Website

A personal portfolio website built with clean HTML, CSS, and vanilla JavaScript. Designed with an editorial/brutalist aesthetic (Vogue-magazine inspired).

## 📁 Website Structure
- `index.html`: Homepage with hero section and tagline.
- `about.html`: Detailed biography, university info, and hobbies.
- `projects.html`: showcase of research and upcoming works.
- `resume.html`: Resume viewer and download link.
- `style.css`: Global styles (Cream/Sage palette, Editorial typography).
- `script.js`: Interactions (Dark mode, Scroll animations, Active links).
- `assets/`: Folder for images and documents.

## 🚀 How to Use / Edit This Website

### 1. Personalize Your Content
- **Text**: Open any `.html` file in a code editor. Look for comments like `<!-- EDIT -->` or simply find the text content to change your bio, university, or project details.
- **Colors**: Open `style.css` and modify the variables in `:root` (e.g., `--bg-color` and `--accent-color`).
- **Links**: Update the `href` attributes in the navigation and footer sections to point to your actual social media profiles.

### 2. Update Assets
- **Profile Photo**: Save your photo as `1.jpeg` and place it inside the `assets/` folder.
- **Resume**: Save your resume as `Riddhi Shobhavat Resume Uni.pdf` and place it inside the `assets/` folder.
- In `index.html` and `resume.html`, the code is already prepared to look for these filenames.

### 3. Deploy to GitHub Pages
1. Push this folder to a new GitHub repository.
2. Go to **Settings** > **Pages** in your repository.
3. Under **Build and deployment**, select the `main` branch and the `/ (root)` folder.
4. Click **Save**. Your website will be live in a few minutes!

## 🌙 Features Included
- **Dark Mode**: Automatically saves your preference in the browser.
- **Scroll Animations**: Sections fade in as you scroll down.
- **Mobile Responsive**: The layout adjusts for tablets and phones.
- **Editorial Design**: Custom typography focus using Google Fonts (Cormorant Garamond, Inter, Playfair Display).
